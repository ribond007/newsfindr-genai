# NewsFindr – Generative AI News Analysis System

## Overview
NewsFindr is a Generative AI-based system designed to analyze and summarize news articles using transformer-based language models.

## Tech Stack
- Python
- Hugging Face Transformers
- LLMs
- NLP preprocessing pipeline

## Features
- Automated news summarization
- Context-aware text understanding
- Prompt optimization for improved output relevance

## Objective
To build an intelligent system capable of generating high-quality summaries from raw news content using modern Generative AI techniques.
